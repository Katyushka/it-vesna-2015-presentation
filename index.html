Цель работы

  • Разработка сайта, содержащего краткие руководства по технологиям программирования.

Задачи

  • Изучить современные и актуальные технологии для разработки WEB-приложений на Java
  • Изучить возможности современных облачных решений для хостинга WEB-приложений на Java
  • Изучить современные технологии и подходы к реализации "frontend" части приложения с Java "backend"-ом
  • Использовать полученные знания для реализации небольшого WEB-приложения - TutorialsWorld

Apache maven

  • Разрешение java зависимостей
  • Сборка приложения с разными профилями
    •             develop - стартует на машине разработчика и
                поднимает in-memory-db HSQLDB
            openshift - стартует на сервере и
                работает с MySql

Основные компоненты приложения

  • Spring - Dependency Injection and Inversion of Control контейнер
  • Spring boot - Автоконфигурирование приложения, упаковка в jar с embedded Tomcat
  • Spring data jpa - Имплементация DAO слоя приложения
  • Spring security - предоставляет механизмы аутентификации и авторизации
  • Spring boot testing - Функциональное и модульное тестирование

Evolution of Web Development

AngularJS:

  • MVC
  • Двусторонняя связь данных
  • Инъекции зависимостей (Dependency Injection, DI)
  • Приложения на одну страницу (Single Page Application, SPA), управление состоянием и Ajax (HTTP)

Мы использовали следующие директивы:

  • ngResource - позволяет удобно работать с REST API
  • ngRoute - добавляет в приложение routing
  • routeStyles - позволяет при роутинге динамически возвращать стили
  • ngSanitize - возвожность предварительной компиляции в html
  • hljs - Подсветка кода
  • textAngular - WYSIWYG редактор
